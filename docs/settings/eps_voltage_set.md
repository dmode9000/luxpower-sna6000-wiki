# EPS Voltage Set

## Призначення

Цей параметр відповідає за встановлення рівня вихідної напруги змінного струму (AC) на порту резервного живлення, коли інвертор працює в автономному режимі (Off-grid/EPS) під час відключення зовнішньої електромережі,.

## Діапазон значень

- Мінімум: 120В (120Vac).
- Максимум: 240В (240Vac).

## Рекомендовані значення

- Для квартири: 230В (сучасний офіційний стандарт напруги для України та Європи).
- Для будинку: 230В.
- За замовчуванням: 230В (230Vac).

## Примітки

- **Важливі обмеження:** Це налаштування впливає на генерацію напруги **тільки** під час відсутності загальної електромережі (блекаут). Коли зовнішня AC мережа підключена і стабільна, інвертор працює в режимі транзиту (байпасу) і видає на резервне навантаження ту саму напругу, яка надходить із зовнішньої мережі.
- **Коли змінювати**: Змінювати це значення потрібно лише в тому випадку, якщо ви встановлюєте інвертор у країні з іншим стандартом мережі, або якщо у вас є специфічне чутливе обладнання, яке суворо вимагає живлення за старим стандартом (наприклад, рівно 220 В). Для абсолютної більшості випадків залишайте значення за замовчуванням.
