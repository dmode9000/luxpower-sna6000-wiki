# Max. AC Input Power(kW)

Налаштування **`Max. AC Input Power(kW)`** відповідає за **обмеження потужності заряду батареї, при заряді від загальної електромережі (Grid)**, щоб заряд батареї не призводив до перевищення заданого порогу потужності на вході в інвертор.

### **Як це працює та для чого використовується:**

Коли зовнішня мережа доступна, інвертор використовує її для двох задач одночасно: живить транзитом споживачів на виході (через вихід EPS/Load) та бере енергію для заряджання акумуляторів. Налаштування `Max. AC Input Power` **не** встановлює жорстку "стелю" для сумарної потужності, яку інвертор братиме з мережі для обох цих потреб. Натомість інвертор стежить за загальним споживанням на вході і динамічно "жертвує" струмом заряду батареї, знижуючи його аж до нуля, щоб спробувати втримати загальне навантаження на вхідну мережу в межах заданого вами ліміту.

1. **Захист вхідної лінії (запобігання вибиванню автоматів):** Це корисне налаштування, якщо на ваш будинок чи об'єкт виділена обмежена електрична потужність, або якщо стоїть слабкий вхідний автоматичний вимикач. Встановивши цей ліміт, ви зменшите вірогіднісь перевантажити зовнішню лінію внаслідок надмірної потужності заряду батареї. Але контроль потужності споживачів на виході є відповідальністю користувача.
2. **Пріоритезація:** Якщо ви увімкнете потужні прилади в будинку, і сумарне споживання (навантаження будинку + зарядка батареї) почне перевищувати встановлений вами ліміт, інвертор миттєво та автоматично зменшить потужність заряджання акумулятора. Це робиться для того, щоб вкластися в задані рамки потужності та не відключити вхідний автомат.

> [!Note] ✍️
> Цей режим не є повноцінним грід-пік-шейвінгом, він не обмежуватиме споживання на вході домішуючи з батареї, якщо ви навантажите вихід понад поріг `Max. AC Input Power`, він впливає лише на потужність заряду батареї.

Якщо ви налаштовуєте систему через LCD-дисплей самого інвертора, це налаштування знаходиться під індексом 35.
