# PV Input Mode

## Призначення

Налаштування `PV Input Mode` визначає фізичну топологію підключення сонячних панелей до інвертора. Воно вказує алгоритмам контролера (MPPT), як саме вони мають відстежувати точку максимальної потужності: працювати як два повністю незалежні канали, об'єднати зусилля для роботи з одним потужним запаралеленим масивом, або взагалі вимкнути пошук MPPT при роботі від стабільного DC джерела.

## Діапазон значень

- `4: Two MPPT connects to different string`: Два незалежні стрінги. Кожен MPPT-трекер працює автономно та шукає власну точку максимальної потужності.
- `3: Two MPPT connects to same string`: Паралельне підключення. Обидва MPPT-трекери програмно та апаратно об'єднуються для роботи з одним потужним стрінгом (масивом), що підключений на обидва входи одночасно.
- `0: DC source mode`: Режим джерела постійного струму. У цьому режимі інвертор не відстежує точку максимальної потужності (MPPT вимкнено), оскільки очікується стабільна напруга на вході.
- За замовчуванням: `4: Two MPPT connects to different string`

## Рекомендовані значення

- Для стандартних систем (різні скати даху, різна кількість панелей, або просто два звичайні стрінги): `4: Two MPPT connects to different string`.
- Для потужних масивів з великим струмом: `3: Two MPPT connects to same string`. Використовується, якщо ви об'єднали панелі паралельно, їхній сумарний струм перевищує ліміт одного MPPT (наприклад, понад 17А), і ви фізично розгалужили цей масив на обидва входи інвертора (PV1 та PV2), щоб рівномірно розподілити струмове навантаження.

## Примітки та обмеження

> [!WARNING] Категорична заборона для систем з кількох інверторів: У паралельних системах (коли 2 або більше інверторів працюють разом на одне навантаження) категорично заборонено підключати один загальний масив сонячних панелей до різних інверторів. Кожен інвертор у паралельній системі повинен мати свої власні, незалежні сонячні панелі.

> [!WARNING] Використання режиму DC: Не обирайте режим DC (DC source mode) при роботі з реальними сонячними панелями. У цьому режимі інвертор не зможе адаптуватися до змін освітленості та температури панелей, що призведе до падіння ефективності генерації.
