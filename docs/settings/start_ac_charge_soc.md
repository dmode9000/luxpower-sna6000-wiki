# Start AC Charge SOC

## Призначення

Цей параметр (також зустрічається як **AC Charge Start SOC**) визначає нижній поріг рівня заряду акумуляторної батареї (у відсотках), при падінні до якого інвертор автоматично почне примусово заряджати її від загальної електромережі змінного струму, якщо в параметрі умов заряду (`AC Charge Based On`) обрано логіку за відсотком: `SOC` або комбіновану `SOC and Time` (разом з виставленими часовими проміжками).

## Доступ

| installer web | end-user web | mobile app | Display |
| :-----------: | :----------: | :--------: | :-----: |
|               |              |            |   16    |

## Діапазон значень

- Мінімум: 1%.
- Максимум: 90%.
- Крок: 1%.

## Рекомендовані значення

- Підберіть значення згідно ваших потреб. Наприклад: якщо потрібно щоб заряд від мережі починався при зниженні SOC до 80% та нижче, встановіть 80% у `Start AC Charge SOC`.

## Обмеження:

- Це налаштування буде працювати **тільки** за умови, якщо в параметрі умов заряду (`AC Charge Based On`) обрано логіку за відсотком: `SOC` або комбіновану `SOC and Time`(разом з виставленими часовими проміжками). Щойно акумулятор розрядиться до встановленого вами рівня `Start AC Charge SOC`, інвертор почне заряджати батарею від зовнішньої мережі доти, поки вона не досягне верхнього порогу — `Stop AC Charge SOC`.
- Не може бути вищим за 90%.

## Налаштування з дисплею

На РК-дисплеї інверторів серії SNA це налаштування доступне в меню під індексом **16**. Під час конфігурації AC Charge, обравши опцію `SOC`, ви зможете задати початковий поріг (`Start Soc`). Проте, найзручніше контролювати цей параметр веб-моніторинг LuxCloud у розділі Charge Setting.
