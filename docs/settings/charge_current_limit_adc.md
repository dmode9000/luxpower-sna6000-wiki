# Charge Current Limit (Adc)

## Призначення

Цей параметр (на РК-дисплеї позначається як **Ic: Maximum charge current**) встановлює максимально допустимий струм (в Амперах), яким інвертор може заряджати акумуляторну батарею. Це глобальний ліміт для всієї системи, що обмежує сумарний струм заряду від усіх доступних джерел живлення одночасно (Сонце + Мережа змінного струму + Генератор).

## Доступ

| Installer Web | End-User Web | Mobile App | Display (LCD) |
| :-----------: | :----------: | :--------: | :-----------: |
|      ✅       |      ?       |     ?      |     ✅ 06     |

_(На дисплеї інвертора загальний ліміт струму заряду знаходиться під індексом **06**)_.

## Діапазон значень

- **Мінімум:** 0 А.
- **Максимум:**
  - Для моделі SNA5000: 110 А.
  - Для моделі SNA6000: 140 А.
- **Крок:** 1 А.
- **За замовчуванням:** Максимально можливий струм для вашої моделі (110 А або 140 А).

## Рекомендовані значення

- Струм заряду безпосередньо залежить від ємності та типу вашого акумулятора. Базове правило безпечного заряду (особливо для свинцево-кислотних АКБ) становить **0.1C**, де C — номінальна ємність батареї. Наприклад, для збірки ємністю 100Ah рекомендований струм становить не повинен перевищувати 10А.
- Для сучасних літієвих батарей безпечним вважається струм до **0.5C** (наприклад, 50А для батареї 100Ah), якщо інше не вказано в інструкції виробника АКБ.

## Примітки та важливі обмеження

> [!WARNING] **Підпорядкування командам BMS:**
> Якщо у вас літієва батарея з кабелем комунікації, інвертор автоматично зчитує максимально дозволений струм заряду (CCL) безпосередньо з плати BMS акумулятора. Ви маєте право встановити в налаштуваннях інвертора **менше** значення, ніж передає BMS, проте ви фізично не зможете змусити інвертор перевищити системний ліміт, який диктує сама батарея.

> [!NOTE] **Взаємодія з іншими лімітами (Мережа / Генератор):**
> Параметр `Charge Current Limit` має найвищий пріоритет. Струм заряду від мережі (`AC Charge Battery Current`) або струм заряду від генератора (`Generator Charge Battery Current`) є підпорядкованими налаштуваннями. Вони повинні бути меншими або дорівнювати значенню `Charge Current Limit`. Навіть якщо ви встановите струм заряду від генератора на рівні 100 А, але загальний `Charge Current Limit` стоїть на рівні 50 А, інвертор заряджатиме батарею струмом не більше 50 А.

## Коли змінювати:

- Налаштовуйте цей параметр під час першого запуску системи, щоб обмежити зарядний струм до безпечних значень згідно документації від виробника, особливо якщо ви використовуєте свинцево-кислотні акумулятори або літієві "самозбірки" без кабелю комунікації.
- Зменшуйте його, якщо ви свідомо хочете "розтягнути" процес її заряджання та заряджати акумулятор меншими струмами (наприклад, 20А або 30А), щоб зменшити теплове навантаження на систему та подовжити термін служби комірок.
