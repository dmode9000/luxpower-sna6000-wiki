# Set Composed Phase

## Призначення

Визначає, яку саме фазу (R, S або T) генеруватиме конкретний інвертор при роботі у складі трифазної паралельної системи. Це дозволяє системі зсунути синусоїду кожного інвертора на 120 градусів для створення правильної трифазної напруги в автономному режимі.

## Діапазон значень

Вибір із фіксованих станів:

- `Phase R` (Фаза 1)
- `Phase S` (Фаза 2)
- `Phase T` (Фаза 3)

## Рекомендовані значення

- За замовчуванням: Інвертор автоматично визначає підключену фазу за наявності зовнішньої мережі.

## Примітки

- **Важливі обмеження:** Змінити це налаштування можна лише тоді, коли інвертор переведено в режим очікування (Standby) або він перебуває у стані помилки (Fault)
- **Коли змінювати:** Це налаштування вимагає ручного втручання інсталятора виключно під час монтажу трифазної системи в повністю автономних умовах (off-grid), коли на об'єкті фізично відсутня зовнішня електромережа. Якщо ж зовнішня електромережа (Grid) доступна, інвертори автоматично виявлять вхідні фази і самостійно призначать собі ролі (R, S або T), тоді ручне налаштування не потрібне.
- **При невірному виборі:** Якщо виявляється невідповідність між тим, на яку фазу програмно налаштований інвертор, і тим, до якої фази він фізично підключений, система заблокує роботу і видасть попередження [`Warning 19`](/troubleshooting/warnings) чи помилку [`Fault 11`](/troubleshooting/faults#fault-11).
